# Cli-Dictionary
Small but handy Command Line dictionary application.

# How to Build this?
1. Clone/Download it
2. Download Rust
3. Run `cargo build --release` Inside the Cli-Dictionary folder
4. You should find your executable in Cli-Dictionary > target > release
